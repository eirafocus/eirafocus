### üåø Focus on What Matters
EiraFocus is designed to bring you peace of mind without the noise. No accounts, no cloud syncing, and no distractions. Just you and your practice.

*   **Guided Breathing:** Equal, Box, and 4-7-8 techniques with fluid visual guidance.
*   **Custom Cycles:** Build and save your own breathing patterns.
*   **Mindfulness Journeys:** Guided meditations including Body Scan, Metta, and Vipassana.
*   **Progress Tracking:** Local session history, daily streaks, and statistics.
*   **Premium Design:** Modern Glassmorphism UI with full Light/Dark mode support.

### üöÄ Getting Started

#### Prerequisites
- Flutter SDK (Stable)
- Android SDK & Emulator/Device

#### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/eirafocus/eirafocus.git
   cd eirafocus
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Audio Setup (Optional):**
   To enable ambient sounds, place your preferred `rain.mp3` and `bell.mp3` in:
   `assets/sounds/`

4. **Run the app:**
   ```bash
   flutter run
   ```
